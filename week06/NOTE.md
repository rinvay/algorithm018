# 第 6 周学习笔记

## 第 12 课 动态规划

分治，回溯，动态规划，本质上都是递归。



共性：寻找重复子问题

差异性：有无最优子结构，中途可以淘汰次优解



动态规划关键点：

1. 最优子结构 opt[n] = bestOf(opt[n-1], opt[n-2]) ，也就是分解子问题，通过子问题的最优解构建全局的最优解
2. 存储中间状态：opt[i]， 也就是找到正确的数据表达方式
3. 递推公式（又叫：状态转移方程/DP方程）

```
eg:
Fibnacci: opt[n] = opt[n-1] + opt[n-2]
二维路径问题：opt[i, j] = opt[i+1, j] + opt[i, j+1]
```
